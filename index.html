ChessAI Engine

A sophisticated chess engine written in C#, utilizing advanced data structures and search algorithms.

C# Bitboards Minimax Alpha-Beta Pruning
Download Release View on GitHub ‚Üó

Technical Implementation

This engine focuses on speed and search efficiency. Instead of simple arrays, it uses Bitboards to represent the board, allowing for extremely fast move generation using bitwise operations.

Search & Optimization

  • Minimax with Alpha-Beta Pruning: Dramatically reduces the number of nodes evaluated.
  • Transposition Tables: Implemented using Zobrist Hashing to store and reuse evaluation results of previously seen positions.
  • Move Ordering: Uses MVV-LVA (Most Valuable Victim - Least Valuable Aggressor) and Killer Moves to ensure the best moves are searched first, maximizing pruning.
  • Iterative Deepening: Allows the engine to return the best found move within a specific time limit.

How to Run

  1. Download the source code or release.
  2. Locate the folder Windows_builds.
  3. Run ChessAI.exe.

Doodle Guesser

A deep learning application that identifies hand-drawn doodles in real-time.

Python TensorFlow Keras Flask
Download Model View on GitHub ‚Üó

Deep Learning Architecture

The core of this project is a Convolutional Neural Network (CNN) trained on a subset of the Google "Quick, Draw!" dataset. The model is capable of recognizing 35 distinct categories with 94.10% test accuracy.

Key Features

  • Large Scale Training: Trained on over 100,000 images using data augmentation techniques.
  • Real-time Inference: Users draw on an HTML5 Canvas, and the image is pre-processed and sent to a Python Flask backend for prediction.
  • Canvas Pre-processing: Includes custom logic to normalize and resize user input to match the 28x28 grayscale training format.

Tech Stack

Developed using Python (Keras/TensorFlow) for the AI components and Flask to bridge the gap between the model and the front-end interface.
