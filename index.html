ChessAI Engine

C++ engine capable of 2M nodes/sec.

Download Release

Project Overview

This is a high-performance Chess Engine written in C++. It utilizes advanced bitwise operations for board representation and implements a Minimax algorithm enhanced with Alpha-Beta pruning.

Key Features

  • Bitboard representation for speed.
  • Move ordering (MVV-LVA).
  • Transposition tables using Zobrist Hashing.
